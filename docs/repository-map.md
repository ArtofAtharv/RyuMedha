# Repository Map

- `/bot`: Node.js WhatsApp bot implementation.
- `/production-bot`: Production configurations for the bot.
- `/testing-bot`: Testing configurations for the bot.
- `/web`: Next.js 15+ frontend application.
- `/web-clone`, `/web-legacy`: Alternative/older frontend iterations.
- `/supabase`: Contains SQL schemas (e.g., `ULTIMATE_CONSOLIDATED_SCHEMA.sql`), migrations, and edge functions.
- `/docs`: Generated technical documentation.
