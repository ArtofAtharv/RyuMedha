# Setup

## Local Setup

1. Clone the repository.
2. Ensure Node.js and npm are installed.
3. For the web app, run:

```bash
cd web
npm install
npm run dev
```

4. For Supabase, install the CLI and configure your project.
5. Configure environment variables for Supabase and WhatsApp API credentials.

## Project layout

- `web/` — Production Next.js app.
- `supabase/` — Database and Edge Functions.

## Notes

- Use `package-lock.json` and `package.json` in the root for dependency management.
- Keep the repo focused on web, supabase, and documentation only.
